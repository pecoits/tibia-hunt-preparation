# Regras do Projeto

- Implemente as alteracoes solicitadas pelo usuario normalmente.
- Antes de fazer qualquer commit ou criar PR, apresente um resumo objetivo do que foi aplicado.
- Depois do resumo, pergunte ao usuario se prefere que seja feito commit e PR, ou somente o commit.
- Nao faca commit nem crie PR antes dessa confirmacao explicita.
